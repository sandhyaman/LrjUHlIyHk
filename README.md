# Python膳食健康系统

## 核心功能
- 用户登录注册
- 食物成分查询
- 食谱信息展示
- 互动问答功能
- 食谱搜索
- 食谱信息推荐
- 用户评论
- 食谱收藏
- 食谱信息管理
- 食物成分管理
- 膳食记录
- 健康分析
- 互动问答管理
- 用户管理

## 适用场景
- 个人健康管理
- 饮食规划
- 健康饮食指导
- 膳食搭配建议

## 技术特点
- 基于Django框架
- 使用MySQL数据库
- 前端采用Vue技术
- 源码附带1.2万字文档
